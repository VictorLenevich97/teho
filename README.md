## Специальное программное обеспечение поддержки принятия решений по техническому обеспечению

### [Сылка на проект в Redmine](https://redmine.rit/redmine/projects/teho)

[![Quality gate](https://sonarqube.rit/api/project_badges/quality_gate?project=va.rit.teho%3Ateho)](https://sonarqube.rit/dashboard?id=va.rit.teho%3Ateho)

В данном репозитории находится backend-часть проекта.

**Стек:** 
* **Spring Boot**
* **Spring Web MVC**
* **Spring Data JPA**
* **Spring Bean Validation**
* **Apache POI**
* **H2 Database** (из-за особенностей сборки в self-contained десктоп-приложение).

В проекте используется Java 8.
